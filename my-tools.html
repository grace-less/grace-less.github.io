My Tools

All these tools except my laptop are new to me to some degree. I’m most familiar with HTML and CSS but only barely. Bootstrap is an entirely new tool to me that has frustrated me throughout this whole process. VScode on the other hand has been a huge relief since the dark background and color-coded text allows me to work on things for extended periods of time. Github is still a bit of a mystery to me, I understand how to use it, I understand its function to a certain extent but I have a lot of technical questions that are irrelevant for being able to use it so I will dive further into it later. -SGD

  • HTML – the basic building blocks of a website such as structure and text
  • CSS – the basic nuances such as colors and spacing
  • Bootstrap – A tool for smoothly making things more dynamic and responsive
  • My laptop – an Acer laptop with Windows 10 and a part of my soul
  • VScode – a text editing program that is headache friendly
  • Github – a website that allows its users to work on various types of documents necessary for web development
